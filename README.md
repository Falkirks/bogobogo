bogobogo
========

> Simple implementation of Bogobogosort


## Install

```sh
$ npm install --save bogobogo
```


## Usage

```js
var bogobogo = require('bogobogo');

bogobogo.create([66, 22, 3, 5, 1, 19]).then(function(result){
	console.log(result);
}).start();
```
